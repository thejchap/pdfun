Using the Fetch API

The fetch() method, available in browsers and Node.js, starts the process of fetching a resource from the network. It returns a promise that resolves to a Response object. This document is a small synthetic article in the spirit of an MDN page — used as a real-world stress test for pdfun's HTML/CSS pipeline.

A basic example

The simplest call to fetch() takes a single argument — the URL of the resource you want to fetch — and returns a promise containing the response.

const response = await fetch("https://example.com/data.json");
const data = await response.json();
console.log(data);

Handling errors

The promise returned by fetch() only rejects on network failure; HTTP error statuses such as 404 or 500 still resolve. You must check the ok property explicitly:

if (!response.ok) {
  throw new Error(`HTTP ${response.status}`);
}

Common request options

  • method — HTTP verb (default GET)
  • headers — request headers as an object or Headers instance
  • body — request body for non-GET requests
  • credentials — controls cookie inclusion
  • signal — an AbortSignal for cancellation
