pdfun

build passing python 3.12+ alpha

A pure-Rust HTML/CSS to PDF renderer with Python bindings. No system dependencies — drop it into a virtualenv and ship.

Installation

uv add pdfun
# or:
pip install pdfun

Quick start

from pdfun import HtmlDocument

doc = HtmlDocument(string="<h1>Hello, pdfun!</h1>")
doc.write_pdf("hello.pdf")

Features

  • CSS 2.1 box model: margins, padding, borders, border-radius, box-sizing
  • Typography: font-family, weight, style, line-height, decoration
  • Paged media: @page, page-break-*, orphans/widows
  • Tables, lists, images (PNG/JPEG)
  • Auto-generated table of contents
